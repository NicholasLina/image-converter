# Image Converter (C# GUI + Rust Core)

Cross-platform desktop image conversion app with:

- **C# GUI (Avalonia)** for drag-and-drop and bulk workflow
- **Rust native core** for format conversion and size estimation
- **Quality slider** for lossy formats (JPEG/AVIF)
- **Estimated output file sizes** before conversion

## Features

- Drag and drop files/folders into the queue
- Bulk add files or recursively import folders
- Convert to:
  - JPEG (`.jpg`)
  - PNG (`.png`)
  - WebP (`.webp`)
  - AVIF (`.avif`)
  - TIFF (`.tiff`)
  - BMP (`.bmp`)
  - GIF (`.gif`)
- Input decode support from common and extended types (via Rust `image` crate), including:
  - PNG, JPEG, WebP, AVIF, TIFF, BMP, GIF, ICO
  - HDR, PNM/PPM/PGM/PBM, DDS, TGA, QOI, EXR
- Per-file status and conversion progress
- Collision-safe output naming (`name.ext`, `name_1.ext`, `name_2.ext`, ...)

## Requirements

- .NET SDK 8.0+
- Rust toolchain (Cargo + rustc, **1.85+ recommended**)

## Build and Run

From the repository root:

```bash
dotnet run --project gui/ImageConverter.Gui/ImageConverter.Gui.csproj
```

The GUI project automatically runs:

```bash
cargo build --release --manifest-path rust-core/Cargo.toml
```

and copies the native library next to the GUI executable.

## Quality and Estimated Size Behavior

- **Quality slider affects JPEG and AVIF** output.
- For formats where visual quality does not apply (e.g., PNG/BMP/GIF/WebP lossless path), the slider is disabled.
- Estimated output size is generated by asking the Rust core to encode in-memory and reporting the resulting byte length.
